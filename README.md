# LTE_scanning
This project is intended solely for educational purposes. Any other use must be reported. 

Hardware:
- VM DragonOS on Virtualbox
- HackRF One
- Device to be tested with an integrated data SIM

The objective of this project is to develop a system for determining the frequency to which an LTE cell is connected, maintaining a stable connection, and calculating signal loss when the target is exposed to a source of interference.
